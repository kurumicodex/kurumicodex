### <img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Animated-Fluent-Emojis/master/Emojis/Hand%20gestures/Waving%20Hand%20Medium-Light%20Skin%20Tone.png" width="30" /> Overview

Saya adalah seorang pengembang web penuh waktu (*fullstack*) yang berfokus pada pembangunan ekosistem digital yang responsif, estetis, dan memiliki performa tinggi. Berbasis di **Surabaya, Indonesia**, saya menggabungkan logika arsitektur backend dengan keindahan antarmuka frontend, sembari menyelaraskannya dengan strategi serta analisis bisnis digital.

Sebagai seorang yang senang bekerja secara mendalam dan fokus, saya menemukan kepuasan dalam memecahkan masalah teknis yang rumit—mulai dari optimalisasi baris kode, manajemen infrastruktur server awan, konfigurasi rute DNS, hingga eksplorasi aspek keamanan sistem.
